O que posso criar com...

HTML

Com HTML posso estruturar páginas web com títulos, parágrafos, imagens, links e formulários. Ele é a base de qualquer site na internet, servindo como esqueleto para que o conteúdo tenha acessibilidade.

CSS

Com CSS consigo estilizar os elementos HTML com cores, fontes e espaçamentos, animações e tornar o layout responsivo, então criando uma boa identidade visual agradável em diversos dispositivos.

JavaScript

O JavaScript permite adicionar interatividade ao site. Com ele posso criar sliders, menus dinâmicos, animações e as integrações com as APIs, validação de formulários e experiências ricas no frontend.

Python

Python é uma linguagem versátil com a qual posso automatizar as tarefas, manipular dados, criar bots, scripts, as APIs, e até desenvolver servidores web com frameworks como Flask ou Django.
